
## Short Written Responses (Task 3)

**1. What is one developer tool or website you use regularly? Why?**

I use Postman regularly to test API endpoints before connecting them into the frontend. It saves a lot of back-and-forth debugging when I can confirm the API response looks right first."

**2. If your code isn't working, what's your first step to debug it?**

I would first open the browser console to look for any error messages first. Usually the error points me right to the problem. If it's not obvious, I will add a few console.log() statements to check what values I am actually working with at different points in the code.
